# Data-Analysis-Project
This respository contain the data analysis performed on an HR data to analyze what factors can decide the amount of pay. And what factors correspond to higher salary.

Overview

This project analyzes an HR dataset to identify factors that influence employee salary levels (low, medium, high). The focus is on exploring relationships between key attributes and salary categories using data analysis and visualization.

Objective
Identify features associated with higher salary levels
Compare average values of selected factors across salary ranges
Highlight patterns that may influence salary differences
Data Preparation
Cleaned dataset and handled missing values
Removed irrelevant columns
Prepared data for analysis
Analysis Method
Grouped data by Salary_Range
Calculated average values for selected features
Visualized results using bar charts (Plotly)
Features Analyzed
Satisfaction Level
Time Spent in Company
Number of Projects
Key Insight Approach

Features with higher average values in the high salary category are considered potential indicators of higher pay.

Tools Used
Python
Pandas
Plotly Express
Conclusion

The analysis highlights patterns between employee attributes and salary levels, providing a foundation for further statistical analysis or predictive modeling.
